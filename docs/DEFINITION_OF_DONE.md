# Definition of Done

A feature is considered complete only when all requirements below are satisfied.

## Code

- [ ] TypeScript passes (`npx tsc --noEmit`)
- [ ] ESLint passes (`npm run lint`)
- [ ] No unused code
- [ ] No duplicated business logic

---

## Architecture

- [ ] Follows AI Business Operating System architecture
- [ ] Business logic is not inside UI
- [ ] Uses existing interfaces where applicable
- [ ] New services have a single responsibility

---

## Documentation

- [ ] Architecture updated if needed
- [ ] Roadmap updated if needed
- [ ] Decisions documented when architecture changes

---

## Git

- [ ] Clean commit message
- [ ] Changes committed
- [ ] Changes pushed

---

## Product

- [ ] Solves a real business problem
- [ ] Fits the product vision
- [ ] Does not introduce unnecessary complexity
