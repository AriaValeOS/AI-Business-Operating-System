# Aria OS Modules

## Core

The system is built around independent modules.

Each module has one responsibility.

---

# Dashboard

Purpose

Show everything important today.

---

# Projects

Manage AI creator projects.

Every creator is a project.

---

Stores:

- style
- prompts
- goals
- branding

---

# Brain

Stores:

- system prompts
- memory
- workflows
- AI behaviour

---

# Content

- captions
- hashtags
- publishing queue

---

# Media Studio

- images

- voice
- thumbnails
- assets

# Calendar

Scheduling.

growth

reach

# Automation

# Settings

## Rule

Modules should communicate through clear interfaces.

No module should directly control another module.
